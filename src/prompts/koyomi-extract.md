# Task

これは江戸時代〜現代の【暦表（こよみ）】の見開きスキャン画像です。
2枚の画像として渡されます。
- 1枚目: 左ページ（後半月）
- 2枚目: 右ページ（前半月）

出力 JSON では、必ず 1月〜12月の時系列順に並べ替えてください。

# 文書の構造

各月について以下を抽出してください。
- 月名
- 月盤（月干支 + 局 + 九星）
- 節気欄（通常月2項目、土旺月は3項目）
- 各日（干支と九星）

## 九星
- 単位漢字: 日, 月, 火, 水, 木, 金, 土
- 数字: 1〜9
- 例: 火6, 水3

## 節気
- 通常月: 2項目
- 1月/4月/7月/10月: 3項目（土旺あり）

# 厳守ルール

1. 月は必ず1月〜12月の昇順
2. 各月の日数は月末日まで埋める
3. 干支は60組の順序を守る
4. 不確実な文字は推測しない
5. 土旺は1月・4月・7月・10月のみ
6. JSON 以外は返さない
