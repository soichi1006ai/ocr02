# 厳守ルール

1. months には 1月・8月〜12月 の6ヶ月のみ含める
2. 出力では月番号の昇順（1, 8, 9, 10, 11, 12）に並べる
3. 各月の日数は月末日まで全て埋める
4. 干支は60組の順序を守る
5. 不確実な文字は "[?]" で囲む（推測しない）
6. date は "月/日" 形式（例: "1/5"）、time は "時:分" 形式（例: "16:31"）
7. JSON 以外は返さない
